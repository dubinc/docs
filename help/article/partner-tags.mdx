<Tip>
  This feature is only available on [Business plans and
  above](https://dub.co/pricing/partners).
</Tip>

Partner tags help you organize partners with flexible labels. You can use tags to track partner types, campaigns, outreach status, lifecycle stage, or any other internal classification that helps your team manage the program.

Unlike [partner groups](/help/article/partner-groups), tags do not control rewards, discounts, or referral link settings. Use groups for program configuration, and use tags for flexible organization and filtering.

Common examples include:

- **Source**: `YouTube`, `Newsletter`, `Agency`, or `Customer referral`
- **Status**: `Needs follow-up`, `Contract sent`, or `High intent`
- **Campaign**: `Q2 launch`, `Black Friday`, or `Webinar cohort`
- **Relationship**: `VIP`, `Strategic`, or `Community`

## Adding tags to partners

This can be done from either the partner table or the partner details view.

### Partner table

With the [tag column](/help/article/managing-program-partners#columns) enabled, click the **tag icon button** to add tags to a partner. This will open the modal view for all your created tags.

    alt="Adding partner tags from the partner table"
  />
</Frame>

You can also bulk edit partner tags using the multi-select checkboxes on each partner row, then click **Update tags** at the top of the table.

<Frame>
  <img
    src="/images/dub-partners/partner-tag-table-bulk-edit.png"
    alt="Bulk editing partner tags in the partner table"
  />
</Frame>

### Partner details

In the partner details view, manage tags from the partner's profile side panel by clicking the **Add tag** or the **Manage** button.

<Frame>
  <img
    src="/images/dub-partners/partner-tag-partner-page.png"
    alt="Adding partner tags from the partner details view"
  />
</Frame>

### Tags modal

Both the partner table and partner details view will open the same tags modal, where you can manage all your tags in one place. Any changes you make here will update across the entire program.

From here, you can select any number of existing tags to assign to your partners. Learn more about [managing tags](/help/article/partner-tags#managing-tags).

## Managing tags

With the tags modal open, you'll be able to see all the tags that you've created and the number of partners assigned. Here you can create new tags, search, update, and delete existing tags.

### Adding new tags

To add a new tag, click into the text field and type the tag you'd like to create. If one already exists, you'll see the match below. If one doesn't exist, click **Create new tag** to create one.

<Frame>
  <img
    src="/images/dub-partners/partner-tag-modal-create.png"
    alt="Creating partner tags from the tags modal"
  />
</Frame>

This will create the tag and assign it to the partner you're currently editing. You can create as many tags as you'd like and assign multiple tags to a single partner.

### Updating tags

To update an existing tag, click the **Edit** option from the tag dropdown. This will open a text field where you can update the name of the tag. This will update the tag name for all partners assigned.

<Frame>
  <img
    src="/images/dub-partners/partner-tag-modal-edit.png"
    alt="Editing partner tags from the tags modal"
  />
</Frame>

### Searching tags

Using the text field, search for the tag you're looking for. If there are any matches, you'll see them below. This is especially helpful for programs with a large number of tags.

<Frame>
  <img
    src="/images/dub-partners/partner-tag-modal-search.png"
    alt="Searching partner tags from the tags modal"
  />
</Frame>

### Deleting tags

To delete an existing tag, click the **Delete** option from the tag dropdown. This will remove the tag from all partners assigned, so be sure as this isn't reversible.

<Frame>
  <img
    src="/images/dub-partners/partner-tag-modal-dropdown.png"
    alt="Deleting partner tags from the tags modal"
  />
</Frame>

## Filtering partners by tags

Partner tags can be used alongside other partner filters such as group, status, and location. This makes it easier to segment your partner table for follow-up, reporting, and other workflows.

<Frame>
  <img
    src="/images/dub-partners/partner-tags-filtering-table.png"
    alt="Filtering partners by tags"
  />
</Frame>

For example, you can filter for partners tagged `VIP` who are also in a specific [partner group](/help/article/partner-groups), or partners tagged `Needs follow-up` after an outreach campaign.

You can also use the same filtering when viewing your [program analytics](/help/article/program-analytics).

<Frame>
  <img
    src="/images/dub-partners/partner-tags-filtering-analytics.png"
    alt="Filtering partners by tags in analytics"
  />
</Frame>

## Tags vs. Groups

| Feature                         | Tags                             | Groups                                                 |
| ------------------------------- | -------------------------------- | ------------------------------------------------------ |
| Primary use                     | Flexible internal organization   | Program configuration and segmentation                 |
| Controls rewards                | No                               | Yes                                                    |
| Controls discounts              | No                               | Yes                                                    |
| Controls referral link settings | No                               | Yes                                                    |
| Best for                        | Labels, workflows, and filtering | Rewards, discounts, applications, and partner movement |
