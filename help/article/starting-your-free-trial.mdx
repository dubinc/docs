When you sign up for Dub, you can start a free trial to explore our paid features before committing to a plan.

## Free trial limits

During your free trial, the following limits apply to your workspace:

| Resource                   | Limit  |
| -------------------------- | ------ |
| Links                      | 100    |
| Clicks                     | 5,000  |
| Payouts                    | 10,000 |
| Domains                    | 5      |
| Tags                       | 5      |
| Folders                    | 5      |
| Groups                     | 5      |
| Network invites            | 0      |
| Users                      | 5      |
| AI                         | 100    |
| API rate limits            | 120    |
| Analytics API rate limits  | 2      |
| Partners                   | 100    |

Once your free trial ends, you'll need to upgrade to a paid plan to continue using these features beyond the free plan limits.
