---
title: "List"
"og:title": "dub.events.list - Retrieve a list of events - TypeScript SDK"
description: "Retrieve a list of events for the authenticated workspace. The list will be paginated and the provided query parameters allow filtering the returned events."
---

<Note>
  Analytics endpoints require a [Pro plan](https://d.to/pro) subscription or
  higher.
</Note>

## Arguments

<ParamField body="event" type="string" default="clicks">
  The type of event to retrieve analytics for. Defaults to `clicks`.
</ParamField>

<ParamField body="domain" type="string">
  The domain to filter analytics for.
</ParamField>

<ParamField body="key" type="string">
  The short link slug.
</ParamField>

<ParamField body="linkId" type="string">
  The unique ID of the short link on Dub.
</ParamField>

<ParamField body="externalId" type="string">
  This is the ID of the link in the your database. Must be prefixed with `ext_`
  when passed as a query parameter.
</ParamField>

<ParamField body="interval" type="enum<string>">
  The interval to retrieve analytics for. Available options are `1h`, `24h`,
  `7d`, `30d`, `90d`, `ytd`, `1y`, and `all`.Takes precedence over `start` and
  `end`.
</ParamField>

<ParamField body="start" type="string">
  The start date to retrieve events for. Cannot be earlier than September 22,
  2022.
</ParamField>

<ParamField body="end" type="string">
  The start date to retrieve events for. Cannot be in the future. If not
  provided, defaults to the current date.
</ParamField>

<ParamField body="timezone" type="string" default="UTC">
  The IANA time zone code for aligning timeseries granularity (e.g.
  `America/New_York`). Defaults to `UTC`.
</ParamField>

  The device to retrieve events for.
</ParamField>

  The browser to retrieve events for.
</ParamField>

  The OS to retrieve events for.

  The referer to retrieve events for.

  The tag ID to retrieve events for.

<ParamField body="qr" type="boolean">
  Filter for QR code scans. If `true`, filter for QR codes only. If `false`,
  filter for links only. If `undefined`, return both.
</ParamField>

<ParamField body="root" type="boolean">
  Filter for root domains. If `true`, filter for domains only. If `false`,
  filter for links only. If `undefined`, return both.
</ParamField>
